## Java

### 累乗した値を求める(pow)

	public static double pow(double a, double b)
	a - 基数
	b - 指数


- 例

	class PowTest{
		public static void main(String[] args){
			double a = 5d;
			double b = 2d;
			double result = Math.pow(a,b);
			System.out.println(a + "^" + b +"=" + result);
		}
	}
