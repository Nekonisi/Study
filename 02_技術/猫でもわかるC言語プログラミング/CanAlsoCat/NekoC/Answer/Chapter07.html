第7章　練習問題　解答

1.
（1）　&a
（2）　*
（3）　5

2. 解答例
/* ptrq.c */

#include <stdio.h>

int main()
{
	int a, *ptr;
	double d, *dptr;

	ptr = &a;
	dptr = &d;

	printf("ptr = %p\n", ptr);
	printf("ptr + 1 = %p\n", ptr + 1);
	printf("dptr = %p\n", dptr);
	printf("dptr + 1 = %p\n", dptr + 1);

	return 0;
}

実行結果の例
ptr = 0012FEDC
ptr + 1 = 0012FEE0
dptr = 0012FEE0
dptr + 1 = 0012FEE8

これをみると、ptr+1はptrより4バイト先のアドレスをdptr+1はdptrより8バイト先のアドレスを示しています。つまり、ptrはint型へのポインタであり、dptrはdouble型へのポインタであることに起因しています。
