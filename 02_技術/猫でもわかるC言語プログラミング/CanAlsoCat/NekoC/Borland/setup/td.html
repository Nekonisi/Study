Turbo Debugger 5.5について

今日、C++Builder ほど、すぐれたデバッグ技術を統合した C/C++ 開発環境はありません。しかし、Turbo Debugger は長年に渡り、強力なデバッグ専用ツールとして世界中の C/C++ 開発者に好まれました。このたび、Borland C++ Compiler 5.5 の無償ダウンロードに対するご好評にお応えし、Turbo Debugger の無償ダウンロードを提供いたします。

今回提供する Turbo Debugger により、32 ビットの Windows アプリケーションの実行を制御したり、実行時のプログラム出力、ソースコード、データ構造、変数の値などさまざまな側面を表示させることができます。

Turbo Debugger の主要な機能:

  • C/C++ およびアセンブリ式の評価
  • C/C++ の例外処理
  • プログラムの実行制御
  • CPU レジスタやシステムメモリを含む低レベル処理
  • データ調査
  • ブレークポイントの設定やログの記録
  • Windows メッセージの追跡やブレークポイントの設定

※Turbo Debugger 5.5 は、「あるがまま」で提供されます。ボーランドは、いっさいの保証・サポートを提供しません。また、このツールに関するバグレポートの受付や将来の修正な ども予定されていません。
